# Mid-semester exam is closed book

The mid-semester test is closed book. Formulas cannot be looked up during the exam — they must be recalled from memory under time pressure.

**Implications:** Every lesson must include a "memorise this" section. Quizzes should progressively require writing formulas from recall, not just recognition. The exponential recency update rule, UCB formula, Bellman equation, and discount return formula are high-priority memorisation targets. Lessons should use spaced repetition — key formulas appear in multiple lessons, not just the one where they're introduced.
