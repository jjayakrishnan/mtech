# Mission: Deep Reinforcement Learning

## Why
Pass the BITS Pilani WILP mid-semester exam for AIMLCZG512. The exam is open-book, 2 hours, 30 marks, 4 questions. The goal is to score well enough that the mid-sem is not a risk to the final grade.

## Success looks like
- Can classify any scenario as MAB vs MDP in one sentence with correct justification
- Can run Value Iteration by hand for 1–2 iterations given a transition table
- Can apply the exponential recency-weighted average update (α-based) and UCB formula numerically
- Can trace through a First-Visit Monte Carlo episode to compute returns and update Q values
- Can explain RL vs Supervised Learning, Reward vs Value, episodic vs continuing, γ effects — in 1–2 crisp sentences each

## Constraints
- WILP student: full-time professional, limited daily study time
- Exam is **closed book** — formulas must be memorised, not just looked up
- Starting from "very high level" — knows actions give rewards, nothing more precise yet
- Mid-semester is **closed book** — formulas must be memorised, not just understood
- Mid-semester scope: Modules 1–2 (RL intro, MAB, Finite MDP, Value Iteration, Monte Carlo)

## Out of scope (for now)
- Deep RL (neural networks, DQN, policy gradient) — these are comprehensive exam topics
- MCTS, AlphaGo, imitation learning — same
- Programming / gym environments — this is exam prep, not coding
