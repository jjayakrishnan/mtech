Key Formulas (closed book — memorize)

A*: f(n) = g(n) + h(n)
Admissible: h(n) ≤ h*(n)
Consistent: h(n) ≤ c(n,n') + h(n')
Alpha-Beta prune: α ≥ β
ACO: τij ← (1-ρ)·τij + Δτij
NEAT fitness sharing: f'i = fi / Σ sh(dij)
