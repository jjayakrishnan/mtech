Key Formulas (closed book — memorize)

Bellman Expectation (Policy Evaluation): \[ V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} p(s'|s,a)\bigl[r + \gamma V^\pi(s')\bigr] \] Bellman Optimality (Value Iteration): \[ V_{k+1}(s) = \max_a \sum_{s'} p(s'|s,a)\bigl[r + \gamma V_k(s')\bigr] \] Q-value (Policy Improvement): \[ Q^\pi(s,a) = \sum_{s'} p(s'|s,a)\bigl[r + \gamma V^\pi(s')\bigr] \] Policy Improvement: \[ \pi'(s) = \arg\max_a Q^\pi(s,a) \] Discounted Return: \[ G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \] First-Visit MC Update: \[ V(s) \leftarrow V(s) + \frac{1}{N(s)}\bigl[G_t - V(s)\bigr] \] UCB Action Selection: \[ A_t = \arg\max_a \left[ Q_t(a) + c\sqrt{\frac{\ln t}{N_t(a)}} \right] \] Constant Step-Size: \[ Q_{n+1} = Q_n + \alpha\bigl[R_n - Q_n\bigr] \] Importance Sampling Ratio: \[ \rho_{t:T-1} = \prod_{k=t}^{T-1} \frac{\pi(A_k|S_k)}{b(A_k|S_k)} \]
